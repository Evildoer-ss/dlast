## USAGE

``` txt
usage: launcher.py [-h] [-i INPUT] -p PLIST [-o OUTPUT] [-d] binary

DLAST

positional arguments:
  binary                ipa's executable file

optional arguments:
  -h, --help            show this help message and exit
  -i INPUT, --input INPUT
                        Input cg.json generated by iblessing
  -p PLIST, --plist PLIST
                        Input APP's Info.plist
  -o OUTPUT, --output OUTPUT
                        Not useful at the moment
  -d, --debug           DEBUG
```

example:

``` sh
python3 ./launcher.py -p test/Lark/Payload/Lark.app/Info.plist ./test/Lark/Payload/Lark.app/Lark
python3 ./launcher.py -i ~/tmp/lark/cg.json -p test/Lark/Payload/Lark.app/Info.plist ./test/Lark/Payload/Lark.app/Lark
```
